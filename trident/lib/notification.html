class trident.lib.notification.handler.TridentNotificationHTTPHandler(notification_config: trident.lib.notification.handler.TridentNotificationHTTPHandlerConfig)[source]¶

Bases: object

send_notification(content: Dict[AnyStr, Any])[source]¶

Send the notification using urllib to send the actual request to the destination. Uses the configuration provided by TridentNotificationHTTPHandlerConfig to fill the request. If both content is used and payload/include result is used then we combine the two. The combine operation only supports content that is JSON parseable so if any other Content-Type header is set then we just send the payload that is defined in the configuration.

Parameters

content (Dict[AnyStr, Any]) – JSON parseable content to include in the notification
